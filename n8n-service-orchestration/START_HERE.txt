🏗️ ARCHITECTURE
═══════════════════════════════════════════════════════════════════════════════

User Message
    ↓
[Service Detector] → Detects service from keywords
    ↓
[Parameter Extractor] → Extracts parameters
    ↓
[N8N Webhook] → Calls service workflow
    ↓
[Backend API] → Executes service
    ↓
[Response Formatter] → Formats response
    ↓
Direct Response to User (No Chatbot AI delay)
    
If No Service Match:
    ↓
[Chatbot AI] → Uses database for response
